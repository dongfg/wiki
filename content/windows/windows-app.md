## WinGet 安装基础应用
```
# PowerToys
winget install --id Microsoft.PowerToys --source winget
# Bitwarden
winget install --id Bitwarden.Bitwarden --source winget -i
# VSCode
winget install --id Microsoft.VisualStudioCode --source winget -i
```
