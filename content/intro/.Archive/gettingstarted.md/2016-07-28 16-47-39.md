# Simiki

## Quick Start

### Install

	pip install simiki

### Update

	pip install -U simiki

### Init Site

	mkdir mywiki && cd mywiki
	simiki init
### folder structure
```
|--wiki
    |--content
    |--output
    |--themes
    |--_config.yml
    |--fabfile.py
```
### Generate

	simiki g
